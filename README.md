# Image_upload

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate

# default port of frontedn  serve is 3000
# upload new image by admin at the route
$ http://localhost:3000/

# To Check all upload image at this route
$ http://localhost:3000/about

# default port of frontedn  serve is 3000
#  User Can Check all upload  at the route
$ http://localhost:3000/allimagelist

#all backend are made with Express.js
For detailed explanation on how things work, check out the [documentation](https://expressjs.com/).

#all frontend are made with Nuxt.js
For detailed explanation on how things work, check out the [documentation](https://nuxtjs.org).

#all UI are made with Vuetify
For detailed explanation on how things work, check out the [documentation](https://vuetifyjs.com).

#All API Make Secure with  OWASP
# In this project  this nuxt-security for api security
$ npm add  nuxt-security
```
